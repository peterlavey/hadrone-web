What is it?

Hadrone es un administrador de proyectos javascript implementados con el motor de nodejs, permite ingresar tanto proyectos node como la base de datos mongodb, el principal objetivo es integrar proyectos web con su api y la base de datos para luego ser levantadas con un solo click, también se puede personalizar la forma de levantamiento.

Getting Started

Agrega proyectos subiendo el package.json a la aplicación
Selecciona el modo de inicio y levanta la aplicación.
